## Local Development

To run this project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm start
```

The site will be available at `http://localhost:3000`

## Deployment

This project is deployed on GitHub Pages. The deployment is automated via GitHub Actions - any push to the main/master branch will automatically deploy to GitHub Pages.

## Technologies Used

- **A-Frame** - WebVR framework
- **Three.js** - 3D graphics library
- **HTML5/CSS3/JavaScript** - Frontend technologies
- **Express.js** - Development server (local only)
